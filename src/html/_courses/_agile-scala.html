Agile Scala: Scala Development in an Agile World

Course Outline

  • An introduction to Agile Methods and related ideas (Lean, Kanban, etc.).
  • Focuses on Extreme Programming and Scrum.
  • Covers the technical disciplines needed to be part of the development team on an Agile project.
  • Covers approaches to collaborating with other stakeholder roles (customers/BAs/users/project managers).

Practical work includes an introduction to TDD and an extensive exercise allowing participants to experience iterative, incremental and evolutionary software development.

This course can be delivered using Scala or Java.

Automated Web-App Testing with Scala - 5 days

Upcoming dates for public courses

London, TBC

Course Outline

This course covers the principles and practices required to write and maintain robust suites of functional tests for web-apps.

Knowledge of Scala is essential. Knowledge of Specs2 will be advantageous but is not essential.

Much of the course deals with the issues that need to be resolved in order to make this kind of testing robust and meaningful and demonstrates, through extensive lab work, proven methods of solving these problems.

The course covers the following topics:

  • Testing fundamentals
  • Testing ReSTful APIs
  • Testing simple web-apps (no AJAX / Comet)
  • Test suite robustness
  • Test suite maintainability
  • AJAX / Comet web-apps
  • Optimisation of test suite run times
