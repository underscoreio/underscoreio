Our courses

Core Scala

Level: Beginner
Length: 2 to 3 days

An introductory course aimed at experienced developers who are encountering Scala for the first time.

Learn how to develop idiomatic, maintainable Scala code. Blend object oriented and functional programming to perform elegant, declarative data manipulation. Master powerful concepts such as implicits to produce expressige domain specific languages.

Core Play

Level: Beginner to intermediate
Length: 1 to 2 days

Use your new-found Scala programming skills to build fast, scalable web sites and services.

Learn how play structures web sites using routes, controllers, and actions. Quickly create HTML5 web pages using templates, forms, and Javascript and CSS. Create lightning fast asynchronous APIs using futures and JSON libraries. Use web sockets and chunked encoding to communicate with clients in real time.
