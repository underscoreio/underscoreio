Core Scala

Level: Beginner
Length: 2 to 3 days

An introductory course aimed at experienced developers who are encountering Scala for the first time. This course assumes no previous Scala or Java knowledge.

Learn how to develop idiomatic, maintainable Scala code. Blend object oriented and functional programming to perform elegant, declarative data manipulation. Master powerful concepts such as implicits to produce expressige domain specific languages.

Core Java 8

Level: Intermediate Java
Length: 1 to 2 days

Java 8 brings many new and powerful features to the table, the most significant of which are lambda expressions and the new streams API for collections.

In Core Java 8 we will show you how to master these new functional programming features. Process huge quantities of data with exceptional speed using the new Java 8 streams. Write secure type-safe code with correct use of the new Optional type. Use closures in combination with classic functional programming patterns to increase developer productivity and write shorter and more maintainable code.

Essential Scalaz

Level: Intermediate
Length: 1 to 2 days

Our 1-2 day Essential Scalaz course is for experienced Scala developers who want to take the next step in engineering robust and scalable systems. The course focuses on four key abstractions in the Scalaz library: Functor, Monoid, Applicative, and Monad, and through a series of mini-projects shows how these abstractions apply across a broad range of practical problems.

Essential Lift

Level: Beginner to intermediate
Length: 1 day

From the author ot the Lift Cookbook comes a captivating new course in the pioneering Lift web framework.

Listen with rapt attention as Richard Dallaway takes you on a voyage of discovery through the world of Lift. Learn to build applications using Lift's no-nonsense view-first approach. Rapidly build a fully validated data layer using Lift Record and your choice SQL or no-SQL databases. Integrate seamlessly with client code using Javascript, AJAX, and Comet technologies. With Lift, the sky's the limit!

Essential Play Web Services

Level: Beginner to intermediate
Length: 1 day

Play is a powerful, lightweight web framework that focuses on scalability and developer productivity.

This course concentrates on using Play to create scalabale RESTful web services. Write fast, non-blocking asynchronous code using futures and promises. Efficiently process and validate JSON and XML data. Integrate with third party web services using Play's WS client API. Implement your own streaming services using using websockets and Akka actors. Build your whole API using clean, testable development practices.

Essential Play Web Sites

Level: Beginner to intermediate
Length: 1 day

Play is a powerful, lightweight web framework that focuses on scalability and developer productivity.

This course concentrates on using Play to create web sites using HTML, Javascript, and CSS. Use Play's Twirl templating language to populate web pages with dynamic content. Build HTML5 content using your choice of Javascript, Coffeescript, Less, and/or CSS. Integrate client and server with AJAX and JSON. Write functional tests using Fluentlenium.

Essential Distributed Systems

Level: Intermediate Java or Scala
Length: 2 days

Distributed systems introduce new challenges in building reliable, scalable software. This course teaches the core theoretical principles for constructing distributed systems and invites particpants to build their own systems using Scala or Java and a set of elemental tools. We will cover three approaches -- message queues, distributed caches, and eventual consistency -- and learn the benefits and drawbacks of each. Finally, in a review session, we will bring together all of the principles covered so far to build a non-trivial distributed system in its entirety.
